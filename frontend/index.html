📘 Sobre o Projeto

🧠 AstroClassifier

O AstroClassifier é uma aplicação web desenvolvida para a classificação automática de objetos astronômicos utilizando técnicas de Machine Learning. O sistema foi treinado com dados fotométricos do SDSS em conjunto com o catálogo WISE, permitindo distinguir entre três classes principais:

  • ⭐ Estrelas (STAR)
  • 🌌 Galáxias (GALAXY)
  • ✨ Quasares (QSO)

⚙️ Como funciona

O modelo utiliza magnitudes (u, g, r, i, z) e dados infravermelhos (W1–W4).

  • Entrada manual de dados
  • Upload de CSV
  • Dataset de demonstração
  • Avaliação em lote
  • Matriz de confusão

🧪 Modelo utilizado

Rede neural em PyTorch com:

  • Duas camadas ocultas
  • Função ReLU
  • Dropout

Treinado com dados rotulados e alta performance na distinção entre classes.

🚀 Tecnologias

  • Python (Flask, PyTorch)
  • JavaScript (Chart.js, PapaParse)
  • HTML e CSS
  • Render (deploy)
