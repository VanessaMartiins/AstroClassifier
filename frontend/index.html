🌌 AstroClassifier

Classificação de objetos astronômicos com rede neural

📘 Sobre o Projeto

🧠 AstroClassifier

O AstroClassifier é uma aplicação web desenvolvida para a classificação automática de objetos astronômicos utilizando técnicas de Machine Learning. O sistema foi treinado com dados fotométricos provenientes do Sloan Digital Sky Survey (SDSS) em conjunto com o catálogo WISE, permitindo distinguir entre três classes principais: ⭐ Estrelas (STAR) 🌌 Galáxias (GALAXY) ✨ Quasares (QSO)

⚙️ Como funciona

O modelo utiliza parâmetros observacionais, como magnitudes em diferentes bandas espectrais (u, g, r, i, z) e dados infravermelhos (W1–W4), para realizar a classificação. A aplicação permite:

  • manual de dados
  • Upload de arquivos CSV
  • Uso de um dataset de demonstração
  • Avaliação em lote com cálculo de acurácia
  • Visualização da matriz de confusão
  • Análise gráfica das previsões

    • 🧪 Modelo utilizado

      O classificador foi desenvolvido com PyTorch, utilizando uma rede neural totalmente conectada com:
    • Duas camadas ocultas
    • Função de ativação ReLU
    • Dropout para regularização
      • O modelo foi treinado com dados rotulados e alcança alta performance na distinção entre diferentes tipos de objetos celestes.

    🚀 Tecnologias utilizadas

  • Python (Flask, PyTorch, Scikit-learn)
  • JavaScript (Fetch API, Chart.js, PapaParse)
  • HTML5 e CSS3
  • Render (deploy do backend e frontend)
