🌌 AstroClassifier

Classificação de objetos astronômicos com rede neural

📘 Sobre o Projeto

O AstroClassifier é uma aplicação de Machine Learning desenvolvida para classificar objetos astronômicos em estrelas, galáxias e quasares, utilizando dados reais.

🔭 Objetivo

Aplicar redes neurais para identificar padrões em dados astronômicos e automatizar a classificação de objetos celestes.

⚙️ Funcionamento

  • Dados reais (SDSS + WISE)
  • Rede neural em Python
  • Normalização com StandardScaler
  • Tratamento de dados faltantes com média
  • Backend Flask + frontend interativo

📊 Avaliação

O modelo pode ser testado individualmente ou em lote, com cálculo de acurácia baseado em amostragem aleatória.

🌌 Importância

Com grandes volumes de dados na astronomia moderna, técnicas de Machine Learning permitem acelerar análises e apoiar descobertas científicas.
